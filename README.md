## Ecommerce API Project

Ecommerce API Project with the following Features:

- User - Registration
- User - Login
- User - Logout
- Product - Add
- Product - Update
- Product - Get one
- Product - Get all
- Product - Delete one
- Cart - Add cart items 
- Cart - Update cart items quantity
- Cart - Get cart items by ID 
- Cart - Delete cart items by ID
